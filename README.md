■　動作環境
---------------------------------------------------------------
　Windows 10

■　起動
---------------------------------------------------------------
　SpaceRun01(64-bit).exeにて起動できます。

■　終了
---------------------------------------------------------------
　Escボタンを押すと終了できます。


■　操作方法
---------------------------------------------------------------
【マウス】
　マウスを動かすことによってロケットを動かすことができます。
　ロケットはカーソルの方へ向かって進みます。
